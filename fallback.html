Quick reference
01
Start your service Make sure the application assigned to this domain is running on its port. Portlabel doesn't manage your processes — it only handles the routing.
02
Check registered domains Open Portlabel and use List to see all domains and their current status. sudo portlabel
03
Toggle a domain on or off Use option 5 in the menu to enable or disable a domain without deleting it. Disabled domains show this page.
04
First visit shows a security warning Portlabel uses Caddy's internal TLS. On first visit Chrome will warn you — click Advanced → Proceed. You won't see it again for this domain.
05
Chrome not resolving .local domains? Disable Chrome's async DNS resolver to ensure it reads your hosts file correctly. chrome://flags/#enable-async-dns
