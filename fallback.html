03
CORS — allow origin Because every project gets a clean .local domain with no port, your CORS config is simple. Use the domain below as your allowed origin — no ports, no wildcards needed.
FastAPI / Python
allow_origins=["https://this.local"]
Express / Node
cors({ origin: "https://this.local" })
Spring Boot
@CrossOrigin(origins = "https://this.local")
04
API base URL Call your backend directly by domain — no ports in the URL, just like production. Works from any frontend registered in Portlabel Dev. https://this.local/your-endpoint
05
Manage all dev projects Open Portlabel Dev to register new projects, view reserved ports, control services, or check CORS info for all registered domains. sudo ./devmode.sh
06
Port range Portlabel Dev auto-assigns ports from the reserved range 30000–39999. These ports are never used by self-hosted tools, so there are no conflicts.
